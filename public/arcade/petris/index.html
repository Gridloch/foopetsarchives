Arcade » Petris

Instructions

Rotate and move the falling shapes to create and clear full lines. The goal is to clear as many lines as possible while getting to the next level.
  • Move the descending blocks back and forth - Left and Right Arrow Keys
  • Rotate the blocks clockwise - Up Arrow Key or the 'X' Key
  • Rotate the blocks counter-clockwise - 'Z' Key
  • Immediately drop the blocks - Down Arrow Key
Bonus: Clearing rows that contain a gem square will award bonus points and remove surrounding blocks.
